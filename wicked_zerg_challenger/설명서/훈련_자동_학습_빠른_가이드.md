## ? 빠른 시작

### 1단계: 훈련 실행

```bash
# 방법 1: 배치 파일 실행 (가장 간단)
wicked_zerg_challenger\bat\start_model_training.bat

# 방법 2: Python 스크립트 직접 실행
python wicked_zerg_challenger\run_with_training.py
```

### 2단계: 훈련 종료

훈련을 중지하려면 `Ctrl+C`를 누르세요.

### 3단계: 자동 실행 (자동)

훈련이 종료되면 **자동으로** 다음이 실행됩니다:

```
======================================================================
AUTO-EXTRACTING AND LEARNING FROM TRAINING DATA
======================================================================

[STEP 1] Extracting training data...
[STEP 2] Analyzing training data...
[STEP 3] Learning from training data...
[STEP 4] Saving extracted data...
[STEP 5] Generating report...

======================================================================
EXTRACTION AND LEARNING COMPLETE
======================================================================
```

## ? 생성되는 결과물

### 출력 디렉토리
```
wicked_zerg_challenger/local_training/extracted_data/
```

### 생성 파일
1. **`training_data_YYYYMMDD_HHMMSS.json`** - 추출된 게임 통계
2. **`comparisons_YYYYMMDD_HHMMSS.json`** - 빌드 오더 비교 데이터
3. **`analysis_YYYYMMDD_HHMMSS.json`** - 데이터 분석 결과
4. **`learned_params_YYYYMMDD_HHMMSS.json`** - 학습된 파라미터
5. **`report_YYYYMMDD_HHMMSS.txt`** - 상세 분석 리포트

---

## ? 리포트 내용

리포트에는 다음 정보가 포함됩니다:

- **훈련 통계**: 총 게임 수, 승률, 평균 게임 시간
- **주요 패배 원인**: 상위 10개 패배 원인 분석
- **상대 종족별 성적**: Terran, Protoss, Zerg별 승률
- **학습된 파라미터**: 업데이트된 빌드 오더 타이밍

## ? 수동 실행 (필요시)

자동 실행이 실패하거나 수동으로 실행하고 싶은 경우:

```bash
# 또는 Python 스크립트 직접 실행

```

## ?? 설정

### 학습률 조정

기본 학습률은 0.1입니다. `run_with_training.py`에서 조정 가능:

```python
learned_params = extractor.learn_from_training_data(
    training_data,
    comparisons,
    learning_rate=0.1  # 0.0 ~ 1.0 (기본값: 0.1)
)
```

## ? 확인 사항

### 자동 실행 조건
- 최소 1게임 이상 완료되어야 자동 실행됩니다
- `game_count > 0` 조건을 확인합니다

### 데이터 파일 위치
자동 실행은 다음 파일들을 읽습니다:
- `wicked_zerg_challenger/data/training_stats.json`
- `wicked_zerg_challenger/data/build_order_comparison_history.json`
- `wicked_zerg_challenger/data/training_session_stats.json`

## ? 문제 해결

### 데이터 파일을 찾을 수 없음
```
[INFO] No training data found to extract. Skipping extraction.
```
- **해결**: 훈련을 먼저 실행하여 데이터를 생성하세요

### 자동 실행 실패
```
[WARNING] Failed to extract and learn from training data: ...
```
- **해결**: 수동으로 실행해보세요 (위의 "수동 실행" 참조)
