## ? 모니터링 방법

### 1. 실시간 모니터링

**명령어**:
```bash
python tools\monitor_training_progress.py [간격(초)]
```

**예시**:
```bash
# 5초마다 업데이트
python tools\monitor_training_progress.py 5

# 10초마다 업데이트 (기본값)
python tools\monitor_training_progress.py 10
```

**표시 정보**:
- 전체 통계 (총 게임, 승률, 난이도 등)
- 최근 5게임 결과
- 승률 변화 추적
- 권장 사항

### 2. 학습률 및 승률 확인

## ? 권장 사항

### 즉시 확인
1. **게임 창 확인**: 게임이 정상적으로 진행되는지 확인
2. **로그 확인**: 빌드오더 실행 로그 확인
3. **통계 확인**: 승률 변화 추적

### 단기 모니터링 (1-2시간)
1. **승률 변화**: Easy 난이도로 승률 개선 확인
2. **빌드오더 정확도**: 건설 타이밍 정확도 확인
3. **패배 패턴**: 패배 원인 분석

### 장기 모니터링 (하루 이상)
1. **전체 승률**: 장기적인 승률 추이 확인
2. **난이도 적응**: Adaptive Difficulty 작동 확인
3. **학습 효과**: 빌드오더 파라미터 학습 효과 확인

## ? 개선 기준

### 승률 기준
- **< 10%**: Easy 난이도 유지
- **10-30%**: Medium 난이도 유지
- **30-70%**: Hard 난이도 고려
- **> 70%**: VeryHard 난이도 고려

### 게임 수 기준
- **< 10게임**: 데이터 부족, 계속 모니터링
- **10-50게임**: 초기 패턴 확인
- **50-100게임**: 안정적인 승률 확인
- **> 100게임**: 장기적인 추이 확인
