# 도구 사용 가이드

## 문제 해결

파일들이 `wicked_zerg_challenger/tools/` 디렉토리에 있으므로, 올바른 경로에서 실행해야 합니다.

## 올바른 실행 방법

### 방법 1: wicked_zerg_challenger 디렉토리에서 실행

```powershell
# wicked_zerg_challenger 디렉토리로 이동
cd D:\Swarm-contol-in-sc2bot\wicked_zerg_challenger

# 모니터링 실행
python tools\monitor_training_progress.py

# 학습률 확인
python tools\show_learning_rate.py
```

### 방법 2: 배치 파일 사용 (권장)

```powershell
# 프로젝트 루트에서
cd D:\Swarm-contol-in-sc2bot\wicked_zerg_challenger

# 배치 파일 실행
bat\monitor_training.bat
bat\show_learning_rate.bat
```

### 방법 3: 전체 경로로 실행

```powershell
# 프로젝트 루트에서
python D:\Swarm-contol-in-sc2bot\wicked_zerg_challenger\tools\monitor_training_progress.py
python D:\Swarm-contol-in-sc2bot\wicked_zerg_challenger\tools\show_learning_rate.py
```

## 도구 설명

### 1. monitor_training_progress.py
- **기능**: 게임 훈련 진행 상황 실시간 모니터링
- **사용법**: 
  ```powershell
  python tools\monitor_training_progress.py [간격(초)]
  ```
- **예시**:
  ```powershell
  python tools\monitor_training_progress.py 5  # 5초마다 업데이트
  python tools\monitor_training_progress.py 10  # 10초마다 업데이트 (기본값)
  ```
- **표시 정보**:
  - 총 게임 수, 승리/패배, 승률
  - 평균 게임 시간
  - 현재 난이도
  - 최근 게임 히스토리
  - 승률 변화 추적

### 2. show_learning_rate.py
- **기능**: 학습률 및 빌드오더 파라미터 확인
- **사용법**:
  ```powershell
  python tools\show_learning_rate.py
  ```
- **표시 정보**:
  - 훈련 통계 (총 게임, 승률, 평균 게임 시간 등)
  - 학습된 빌드오더 파라미터 (gas_supply, spawning_pool_supply 등)
