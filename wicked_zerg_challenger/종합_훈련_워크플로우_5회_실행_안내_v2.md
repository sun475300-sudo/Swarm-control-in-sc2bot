## ? 워크플로우 단계

각 반복마다 다음 단계가 순차적으로 실행됩니다:

### 1. 자동 에러 수정
- `tools/auto_error_fixer.py` 실행
- 자동으로 발견된 에러 수정
- 문법 오류, 인덴테이션 오류 등 수정

### 2. 코드 품질 검사
- `tools/code_quality_improver.py` 실행
- 코드 품질 검사 및 개선
- 스타일 통일화

### 3. 게임 훈련
- `run_with_training.py` 실행
- 추가 훈련 세션 실행
- 학습된 파라미터 적용

### 4. 리플레이 빌드오더 학습
- `local_training/scripts/replay_build_order_learner.py` 실행
- 리플레이에서 빌드오더 추출
- 학습된 데이터 자동 적용 (`config.py`를 통해)

### 5. 리플레이 비교 분석
- `local_training/strategy_audit.py` 실행
- 프로게이머 vs 봇 비교 분석
- 시간 오차, 순서 오류, 자원 효율 분석

### 6. 5번 반복
- 위 과정을 총 5회 반복

## ? 예상 소요 시간

- **에러 수정 1회**: 약 5-10분
- **코드 품질 검사 1회**: 약 5-10분
- **게임 훈련 1회**: 약 30-60분 (게임 수에 따라)
- **빌드오더 학습 1회**: 약 10-30분 (리플레이 수에 따라)
- **비교 분석 1회**: 약 10-30분 (리플레이 수에 따라)
- **총 5회 반복**: 약 300-700분 (5-12시간)
