   ```python
           # C++ closer_than() 사용
           return self.bot.enemy_units.closer_than(radius, position)
   ```

2. **수동 거리 루프 교체**:
   ```python
   # BEFORE (느림):
   nearby = [e for e in enemies if e.distance_to(pos) < 15]

   # AFTER (빠름):
   nearby = enemies.closer_than(15, pos)
   ```

```python
    self.logger.error(f"[{self.__class__.__name__}] Value error: {e}")

    if self.debug_mode:
        raise
```

### 4.1 성능 프로파일링 (Task #12) 🔲

**목표**: 프레임 시간 병목 식별

**방법**:
```python
async def on_step(self, iteration):
    frame_start = time.perf_counter()
    timings = {}

    for manager_name, manager in self.managers.items():
        start = time.perf_counter()
        await manager.on_step(iteration)
        timings[manager_name] = (time.perf_counter() - start) * 1000

    total_time = (time.perf_counter() - frame_start) * 1000
    if total_time > 20:  # >20ms
        self.logger.warning(f"Slow frame: {total_time:.2f}ms")
```

**목표**:
- 상위 5개 느린 매니저 식별
