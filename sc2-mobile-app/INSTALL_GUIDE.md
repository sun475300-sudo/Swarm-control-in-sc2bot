# 📱 모바일에서 앱 설치하기

GitHub Monitor 앱을 모바일 기기의 홈 화면에 추가하여 네이티브 앱처럼 사용하세요!

## 🚀 빠른 설치

### 방법 1: 자동 설치 프롬프트 (권장)
앱에 처음 접속하면 **자동으로 설치 프롬프트**가 나타납니다!

1. 앱 접속: https://5174-inm4xsfrf1dh760iaux6u-02b9cc79.sandbox.novita.ai
2. 하단에 나타나는 **"앱 설치하기"** 배너 확인
3. **"설치"** 버튼 클릭
4. 설치 완료! 🎉

---

## 📱 기기별 설치 방법

### Android (Chrome)

#### 자동 프롬프트 사용
1. Chrome 브라우저로 앱 접속
2. 주소창 하단에 **"홈 화면에 추가"** 배너 자동 표시
3. **"설치"** 또는 **"추가"** 버튼 클릭
4. 확인 대화상자에서 **"설치"** 클릭

#### 수동 설치
1. Chrome 브라우저로 앱 접속
2. 우측 상단 **⋮** (메뉴) 버튼 클릭
3. **"홈 화면에 추가"** 또는 **"앱 설치"** 선택
4. 앱 이름 확인 후 **"추가"** 클릭
5. 홈 화면에 아이콘이 추가됩니다!

---

### iPhone/iPad (Safari)

#### 설치 방법
1. Safari 브라우저로 앱 접속
2. 하단 **공유** 버튼 (□↑) 클릭
3. 아래로 스크롤하여 **"홈 화면에 추가"** 선택
4. 앱 이름 확인 후 **"추가"** 클릭
5. 홈 화면에 아이콘이 추가됩니다!

> ⚠️ **iOS 주의사항**: Safari에서만 설치 가능합니다. Chrome, Firefox 등은 지원하지 않습니다.

---

### Android (Samsung Internet)

1. Samsung Internet 브라우저로 앱 접속
2. 하단 **메뉴** 버튼 (三) 클릭
3. **"페이지 추가"** → **"홈 화면"** 선택
4. 앱 이름 확인 후 **"추가"** 클릭

---

### Desktop (Chrome/Edge)

1. Chrome 또는 Edge 브라우저로 앱 접속
2. 주소창 우측에 **⊕** (설치) 아이콘 클릭
3. **"설치"** 버튼 클릭
4. 작업 표시줄 또는 시작 메뉴에 추가됩니다!

## 🔧 설치 문제 해결

### "설치" 버튼이 보이지 않아요
- **Chrome/Edge**: 주소창 우측 확인
- **Safari (iOS)**: 공유 버튼 → "홈 화면에 추가"
- **Samsung Internet**: 메뉴 → "페이지 추가"

### iOS에서 설치가 안 돼요
- ✅ Safari 브라우저 사용 확인
- ❌ Chrome, Firefox는 iOS에서 PWA 설치 불가
- 공유 버튼(□↑)을 찾아서 사용하세요

### 설치 프롬프트가 사라졌어요
- 페이지를 새로고침하세요
- "나중에" 클릭 시 7일 후 다시 표시됩니다
- 브라우저 메뉴에서 수동 설치 가능

### 아이콘이 표시되지 않아요
- 설치 후 몇 초 기다리세요
- 홈 화면을 새로고침하세요 (다른 페이지로 이동 후 돌아오기)
