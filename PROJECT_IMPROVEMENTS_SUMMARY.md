### 2. ? Cross-Platform 설치 스크립트

**작업 내용:**
- Windows용 설치 스크립트 (`setup.ps1`)
- Linux/macOS용 설치 스크립트 (`setup.sh`)
- 설치 가이드 문서 (`SETUP.md`)

**주요 기능:**
- Python 버전 자동 확인 (3.10+)
- 가상 환경 자동 생성
- 의존성 자동 설치
- 환경 변수 파일 자동 설정
- 설치 확인 자동 검증

**사용 방법:**

**Windows:**
```powershell
.\setup.ps1
```

**Linux/macOS:**
```bash
chmod +x setup.sh
./setup.sh
```

**효과:**
- 새 개발자 온보딩 시간 단축
- 설치 과정 자동화
- 크로스 플랫폼 지원
